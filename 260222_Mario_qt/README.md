# Mario Qt

Порт игры Mario OOP на Qt. Исходная консольная версия находится в `260201_marioMulti/mario_OOP`.

## Сборка (MSYS2)

Откройте терминал MSYS2 (например `mingw64.exe`), перейдите в папку проекта и выполните:

```sh
./build.bat
```

Это создаст директорию `build_ninja` и соберёт исполняемый файл с помощью CMake/Ninja и Qt (Qt5/Qt6).

Запустить можно из того же терминала:

```sh
cd build_ninja
./marioqt.exe
```

Игровой цикл и управление реализованы через Qt: A/D — движение, пробел — прыжок.
